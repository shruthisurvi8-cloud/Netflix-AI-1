# Netflix AI SQL Assistant

An AI-powered Streamlit application that converts natural language into SQL queries and executes them on a Netflix dataset using Google Gemini AI.

## Features

- Convert English questions into SQL
- Execute SQL queries on a Netflix database
- Display query results instantly
- Built with Streamlit, SQLite, and Gemini AI

## Tech Stack

- Python
- Streamlit
- SQLite
- Google Gemini API
